## About

Web version of the [FRC 930 training app](https://github.com/FRC930/Training_App) README for easier visibility. 

This site is built using [Hugo](https://gohugo.io), a static site generating tool that converts plain markdown into websites. This site uses the [Hextra](https://imfing.github.io/hextra/) theme for Hugo. Since there are many versions of Hugo, using Hugo version manager simplifies the Hugo update workflow by writing to the Hugo version and edition to a `.hvm` file found in the project root.
